# Archived Project Material

Godot ignores this directory. It contains confirmed-unused or superseded material preserved for provenance, comparison, and possible manual recovery.

## Rules

- Nothing here may be referenced by a runtime scene, resource, active tool, or smoke test.
- Active content and explicitly supported loadable rollbacks stay outside this archive.
- `characters/playable/opaw/review_variants/` contains rejected handless/armless experiments plus their retired builders and tests. The supported Wayfarer rollback remains under `assets/characters/playable/opaw/variants/wayfarer_original/`.
- `characters/playable/awakened_legacy/` contains the superseded Awakened runtime and source material.
- `environment/sanctuary/legacy_service_pass/` and `characters/npcs/legacy_service_pass/` contain the replaced Eira, Orren, building, stall, scene, import, and source files.

Restore archived content only through a reviewed migration that re-establishes canonical paths, documentation, imports, and focused tests.
